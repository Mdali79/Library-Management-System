## Usage

The system is ready to use with Computer Science department data. All new registrations will default to Computer Science department, and all sample data reflects CS-specific content.

## Customization

If you need to add more CS-specific data:
- Edit `database/seeders/CategorySeeder.php` for more categories
- Edit `database/factories/bookFactory.php` for more book titles
- Edit `database/factories/autherFactory.php` for more authors
- Edit `database/factories/publisherFactory.php` for more publishers

## Notes

- All existing functionality remains the same
- Only the sample data has been customized for CS department
- New registrations can still select any role, but will be in CS department
- Books can be added in any category, but default categories are CS-focused
